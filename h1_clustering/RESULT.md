# H1 클러스터링 결과

## 표본 기준

- 수집 대상: 270~290레벨 본캐 후보 중 checkpoint-active 조건을 만족한 캐릭터
- active checkpoint: `2025-06`, `2025-12`, `2026-05`
- 기준 의미:
  - `2025-06`, `2025-12`: 메이플스토리 대형 이벤트가 주로 진행되는 시점
  - `2026-05`: 현재 실험 기준에서 실제로 활동 중인 유저를 확인하기 위한 시점
- 본캐 표본: 2,000명
- 피처 생성 완료 표본: 2,000명
- 클러스터링에 사용 가능한 표본: 1,979명

## 피처 전처리 정책

`feature_selection.ipynb`에서 후보 피처를 다시 평가할 때 다음 전처리 정책을 적용했다.

- `log1p_avg_monthly_delta_cumexp`: 누적 경험치 월평균 증가량에 `log1p`를 적용한 값을 그대로 사용
- 레벨, 유니온, 어센틱, HEXA 조각류 증가량: 하한 `0`, 상한 `p99`로 clipping
- 전투력 증가량: `p01~p99` winsorize
- 최종 선택 피처는 K-Means 적용 전 `StandardScaler`로 표준화

증가량 피처에 하한 `0`을 둔 이유는 감소값이 실제 성장 정체라기보다 장비 교체, 측정 시점 차이, API 스냅샷 변동의 영향을 받을 수 있기 때문이다. 상한 clipping/winsorize는 극단적으로 큰 성장 유저가 군집 중심을 과도하게 끌어가는 것을 막기 위한 처리다.

## 최종 선택 피처

`feature_selection.ipynb`는 family-diverse 조건에서 다음 3개 피처 조합을 최종 선택했다.

| Alias | 실제 사용 값 |
|---|---|
| `cumexp_avg` | `log1p_avg_monthly_delta_cumexp` |
| `union_avg` | `avg_monthly_delta_union_level`, `[0, p99]` clipping |
| `access_ratio` | 월별 접속 비율 |

선택 결과는 다음과 같다.

| 항목 | 값 |
|---|---:|
| K-Means k | 3 |
| Silhouette | 0.5446 |
| Elbow 기준 k | 4 |
| 주차 후보 cluster id | 1 |
| 주차 후보 수 | 412명 |
| 주차 후보 비율 | 20.8% |

기존 후보였던 성장량 중심 피처 조합보다, 누적 경험치 성장 수준, 유니온 성장, 접속 비율을 함께 보는 조합이 군집 분리도와 해석 가능성 측면에서 더 안정적이었다. 특히 `access_ratio`가 포함되면서 단순히 성장량이 낮은 유저와 실제로 지속 접속한 정체 유저를 구분하는 데 도움이 된다.

## 해석

이 결과는 "확정 주차 유저" 라벨이 아니라, 성장 정체 특성이 강한 군집을 식별한 결과로 해석해야 한다. 현재 H1의 역할은 지도학습 정답지를 만드는 것이 아니라, 이후 H2/H3 분석에서 검증할 주차 후보군을 일관된 기준으로 정의하는 것이다.

현재 기준에서 주차 후보군은 다음 특성을 가진다.

- 누적 경험치 월평균 증가량이 상대적으로 낮다.
- 유니온 성장도 제한적이다.
- 단순 미접속자가 아니라 checkpoint-active 수집 조건과 접속 비율 피처를 통과한 활동 표본 안에서 분리된다.

이 기준은 "활동은 하지만 성장 속도는 낮은 캐릭터"를 포착하는 데 초점을 둔다.
