# 가설 2 — 레벨/직업 분포 검정 (Chi-Square)

주차 유저 군집이 특정 레벨 구간 및 직업 계열에 통계적으로 유의미하게 집중되어 있는지 검정한다.

## 입력

H1 클러스터 레이블 + `level_band`, `class_group` (EDA Sec 4/5에서 정의)

## 검정 대상

| 변수 | 구분 |
|---|---|
| 레벨 구간 | `~260` / `260~280` / `280~300` / `300+` |
| 직업 계열 | 전사 / 마법사 / 궁수 / 도적 / 해적 (초보자 제외) |

## 방법

- Chi-Square Test (α = 0.05)
- 기대 빈도 확인 (모든 셀 ≥ 5 조건 충족 여부 — EDA Sec 5에서 사전 확인 완료)

## 출력

- 교차표 및 기대 빈도표
- Chi2 통계량, p-value, 자유도
- 주차 유저 비율이 높은 레벨 구간 / 직업 계열 특정
