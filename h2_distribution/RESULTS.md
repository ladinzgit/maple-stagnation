# H2 H1 후보군 분포 검정 결과

## 설계

- 입력: `data/features_monthly.csv`, `data/cluster_labels.csv`
- 보조 입력: `data/h1_current_candidates.csv`
- 기본 라벨: `is_stagnant_cluster` (H1 K-Means 성장 정체/주차 후보 cluster)
- 보조 라벨: `is_current_parking_candidate`, `is_high_confidence_candidate`
- 사전 정의 범주: 레벨 `270-279 / 280-285 / 286-290`, 직업 계열 `전사 / 마법사 / 궁수 / 도적 / 해적`
- 검정: 카이제곱 독립성 검정, `alpha = 0.05`; 효과크기 Cramer's V; 후보 셀 표준화 잔차
- 보수적 점검: 고정 주변합 Monte Carlo 검정(100,000회), 두 기본 교차표에 대한 Holm 보정

## 기본 라벨 결과: H1 클러스터 후보

H1 클러스터 후보는 412명(20.82%)이다. 분석 표본은 1,979명이다.

## 판정

H1 클러스터 후보 412명 기준으로 레벨 구간 분포는 Holm 보정 후에도 유의해 H2의 레벨 구간 불균형 가설을 지지한다. 후보는 `270-279` 구간에 강하게 집중되고 `286-290` 구간에서는 드물다. 레벨 효과의 Cramer's V는 0.376로, 63명 현재성 라벨에서 보였던 작은 효과보다 훨씬 크다.

반면 직업 계열 분포는 H1 클러스터 후보, 현재성 후보, 고신뢰 현재성 후보 모두에서 유의하지 않다. 따라서 H2의 결론은 `레벨 구간 집중은 확인, 직업 계열 집중은 미확인`으로 정리한다. 이 결과는 H1 파생 후보 라벨의 분포 검정이며 실제 주차 유저 ground truth 검정은 아니다.
